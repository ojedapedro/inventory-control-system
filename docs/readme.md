# Sistema de Control de Inventario

Una aplicación web para el control de inventarios diseñada específicamente para auditores.

## Características

- **Carga de inventario teórico**: Importación desde archivos Excel
- **Toma de inventario físico**: Escaneo de códigos de barras en tiempo real
- **Informe de incidencias**: Comparación automática entre inventario teórico y físico
- **Gestión de datos**: Registro de fecha, tienda y responsable
- **Historial**: Almacenamiento de los últimos 5 inventarios realizados
